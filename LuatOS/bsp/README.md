# BSP目录

本目录存放的bsp, 是还没独立存放的.

已经独立的存放的bsp, 将从本目录移除, 例如air101/103/105等

一些已经停止支持的bsp, 也会移除

更多bsp, 请查阅 https://gitee.com/openLuat
